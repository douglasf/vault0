# Vault0

A terminal UI (TUI) kanban board built with [Ink](https://github.com/vadimdemedes/ink) (React for CLI), [Drizzle ORM](https://orm.drizzle.team/) with SQLite, and [Bun](https://bun.sh/).

Features (planned):
- Hierarchical tasks with parent/child relationships
- Dependency tracking between tasks
- SQLite persistence via Drizzle ORM
- Interactive terminal UI with keyboard navigation

See `.opencode/plans/` for the full implementation plan.
